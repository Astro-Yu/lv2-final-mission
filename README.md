# 농구 코트 예약 서비스

## 간단한 농구 코트 예약 서비스입니다.

- [ ] 농구 코트는 1시간만 예약 가능합니다.
- [ ] 08:00 ~ 24:00 사이로 예약 가능합니다.
- [ ] 이미 예약되어 있는 코트는 예약 불가능합니다.
- [ ] 코트 사정상 업체 쪽에서 예약을 취소할 수 있습니다.
- [ ] 평일 가격은 시간당 10,000원 입니다.
- [ ] 주말 및 공휴일 가격은 시간당 15,000원 입니다.
- [ ] 코트는 최소 1명, 최대 14명까지 예약 가능합니다.
